## Package Documentation

### Core Packages

- **@meridianalgo/core** - Core types, interfaces, and plugin system
- **@meridianalgo/indicators** - 100+ technical indicators
- **@meridianalgo/data** - Data adapters and management
- **@meridianalgo/strategies** - Trading strategy templates
- **@meridianalgo/backtest** - Backtesting engines

### Analysis Packages

- **@meridianalgo/risk** - Risk metrics and monitoring
- **@meridianalgo/portfolio** - Portfolio optimization
- **@meridianalgo/models** - Machine learning models

### Execution Packages

- **@meridianalgo/execution** - Order execution and management
- **@meridianalgo/optimize** - Parameter optimization

### Utility Packages

- **@meridianalgo/utils** - Shared utilities
- **@meridianalgo/cli** - Command-line tools
- **@meridianalgo/visualize** - Charting and visualization
- **@meridianalgo/pipeline** - Workflow orchestration
- **@meridianalgo/compliance** - Audit and compliance

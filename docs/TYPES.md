# Common Types

This document summarizes the core interfaces used across the MeridianAlgo framework.

## Data Types

### `Bar`
Standard representation of a price candle (OHLCV).

```typescript
interface Bar {
  t: Date;    // Time
  o: number;  // Open
  h: number;  // High
  l: number;  // Low
  c: number;  // Close
  v: number;  // Volume
}
```

### `Quote`
Real-time bid/ask data.

```typescript
interface Quote {
  t: Date;
  bid: number;
  bidSize: number;
  ask: number;
  askSize: number;
}
```

## Strategy Types

### `Signal`
Generated by strategies to indicate a desired trade direction.

```typescript
interface Signal {
  t: Date;
  value: number; // +1 (Buy), -1 (Sell), 0 (Exit/Neutral)
  metadata?: Record<string, any>;
}
```

### `Strategy`
Base interface for all trading strategies.

```typescript
interface Strategy {
  id: string;
  init(bars: Bar[]): void;
  next(bar: Bar): Signal | null;
  generate?(bars: Bar[]): Signal[];
}
```

## Backtesting Types

### `Trade`
Details of an executed trade.

```typescript
interface Trade {
  symbol: string;
  entryTime: Date;
  exitTime: Date;
  side: 'long' | 'short';
  entryPrice: number;
  exitPrice: number;
  qty: number;
  pnl: number;
  pnlPercent: number;
}
```

### `BacktestResult`
Summary of a backtest run.

```typescript
interface BacktestResult {
  metrics: PerformanceMetrics;
  trades: Trade[];
  equity: EquityPoint[];
  positions: Position[];
}
```

## Portfolio Types

### `Weights`
Asset allocation representation.

```typescript
type Weights = Record<string, number>; // symbol -> weight (0 to 1)
```
